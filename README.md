# HireLogic Screening Agent

HireLogic Screening Agent is a full-stack interview screening application with a React client and a Node.js service layer. The repository contains a lightweight frontend for the interview flow and a backend service that coordinates screening logic and persistence helpers.

## Tech Stack

- React with Vite
- Node.js backend
- JavaScript

## Repository Structure

- client/: React frontend built with Vite
- server/: Node.js backend service

## Getting Started

### Client

`ash
cd client
npm install
npm run dev
`

### Server

`ash
cd server
npm install
npm start
`

## Environment

Copy the provided example environment files before running locally:

- client/.env.example
- server/.env.example

## Notes

This repository was prepared for public GitHub hosting with secret-safe environment handling. Real local values should remain in untracked .env files only.
